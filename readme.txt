Revert WooCommerce Image Gallery replaces the Woocommerce v2.7 product image gallery with zoom effect to the old WooCommerce gallery.


== Description ==

WooCommerce v2.7 will have zoom effect on the product images. As some merchants may not have uploaded high-resolution images for their products, this will not look good when zoomed.

Revert WooCommerce Image Gallery replaces the Woocommerce v2.7 product image gallery with the old one. This should allow merchants to prepare and upload high resolution images instead of showing blurry images.

Users will still be able to use other new features that come with WooCoomerce 2.7.


== Installation ==

= Install via the WordPress Dashboard =

1.	Login to your WordPress Dashboard.

2.	Navigate to Plugins, and select "Add New" to go to the "Add Plugins" page.

3.	In the search bar, enter "Revert WooCommerce Image Gallery" and hit Enter.

4.	Click install, and wait for the plugin to download. Once done, activate the plugin.


= Install via FTP =

1.	Extract the zip file, Login using your FTP client, and upload the revert-woocommerce-image-gallery folder to your /wp-content/plugins/ directory.

2.	Login to your WordPress Dashboard.

3.	Go to Plugins and activate "Revert WooCommerce Image Gallery" plugin.



== Frequently Asked Questions ==

= Will this work with other ecommerce plugins? =

No, this plugin was coded to work only for WooCommerce.

= Where is the settings page? =

There is no settings page. The Image Gallery should revert to the old version the moment you activate the plugin.
